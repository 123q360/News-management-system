**展示内容：**

**展示内容：**

- 多兴趣用户数（3-4个类别权值>0）

**用户画像标签：**
- 无兴趣 → 新用户/不活跃用户
- 单一兴趣 → 专注型用户

**颜色方案：**
- 无兴趣：灰色
- 单一兴趣：蓝色
- 双兴趣：粉色
- 多兴趣：青色

## 🔧 技术实现细节

### 后端SQL查询

#### 1. 偏好分布统计
```sql
SELECT preference_category, COUNT(*) as count 
FROM user 
GROUP BY preference_category
```

#### 2. 兴趣广度分析
```sql
SELECT 
  CASE 
    WHEN (有权值的类别数) = 0 THEN 'none'
    WHEN (有权值的类别数) = 1 THEN 'single'
    WHEN (有权值的类别数) = 2 THEN 'double'
    ELSE 'multiple'
  END as interest_type,
  COUNT(*) as count
FROM user u
LEFT JOIN user_preference up ON u.preference_id = up.id
GROUP BY interest_type
```

#### 3. TOP10用户
```sql
SELECT 
  u.username,
  u.preference_category,
  up.tech, up.sports, up.game, up.politics,
  (tech + sports + game + politics) as total
FROM user u
LEFT JOIN user_preference up ON u.preference_id = up.id
ORDER BY total DESC
LIMIT 10
```
